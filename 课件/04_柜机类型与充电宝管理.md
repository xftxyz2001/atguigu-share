## 1、柜机类型

### 1.1、前期准备

1、启动nacos

2、启动微服务模块：

```tex
share-gateway、share-auth、share-system、share-device
```

3、配置商品微服务模块网关：

在nacos配置中心：share-gateway-dev.yml文件添加设备服务的路由配置信息

```yaml
        # 设备服务
        - id: share-device
          uri: lb://share-device
          predicates:
            - Path=/device/**
          filters:
            - StripPrefix=1
        # 规则服务
        - id: share-rule
          uri: lb://share-rule
          predicates:
            - Path=/rule/**
          filters:
            - StripPrefix=1
```

4、进入share-ui目录启动前端项目：

```shell
yarn dev
```

5、登录前端项目

http://localhost/login?redirect=/index
